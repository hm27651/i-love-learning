# I Love Learning

I Love Learning 是一款面向个人长期学习的本地化题库与复习平台。它以清晰的知识层级组织课程、认证和自学内容，支持题库管理、分类练习、错题复习、掌握度跟踪、模拟考试、学习计划与实践任务。所有学习数据默认保存在本机，适合在可信家庭局域网中持续使用。

固定知识层级为：

`学习项目 → 科目 → 章节 → 知识点 → 题目`

支持五类题型、间隔复习、掌握度、模拟考试、学习计划、实践任务、项目级准备度，以及 PDF、DOCX、XLSX、CSV 题库导入。

## 启动

1. 双击 `start.bat`。脚本会创建本地运行环境并自动补齐解析依赖。
2. 电脑打开 `http://127.0.0.1:23456`。
3. 手机连接同一 Wi-Fi，使用设置页显示的局域网地址。

电脑必须保持开机，启动窗口也必须保持打开。软件没有登录保护，**禁止配置公网端口映射，也不要在不可信网络中运行**。

## 项目与模块

- 普通刷题：题库、练习、复习、学习进度。
- 考试备考：增加模拟考试、学习计划和准备度。
- 实操认证：再增加实践任务与实践完成率门槛。

每个浏览器独立记住当前项目。归档项目会保留全部数据；永久删除前软件自动在项目根目录的 `backups` 下创建完整备份。

## 知识树安全删除

- 空科目、章节和知识点可确认后直接删除。
- 包含内容的节点必须先迁移到当前项目中的其它同类节点，题目 ID 和学习进度保持不变。
- 同名章节或知识点只有明确确认后才会合并；现有重复名称和运行中的导入任务会阻止删除。
- 非空节点删除前自动备份数据库，最近保留 20 份；轻量操作报告保存在 `backups/knowledge_operation_reports`。

## 导入题库

从“导入中心”上传文件。原文件会永久保存并记录 SHA-256，随后需要确认目标项目和科目。所有成功提交的题目先进入草稿，人工核验后才能进入默认练习与模拟。

- PDF、DOCX：只解析结构化、可复制文本，不做 OCR。
- XLSX、CSV：使用导入中心提供的固定模板。
- DOC、XLS、扫描版 PDF 和未知版式：停止导入并显示异常，不会部分写入。
- 一份文件只能归属一个科目；无法识别章节时进入“待分类”。

外部脚本可使用同一套解析、校验、去重和提交逻辑：

```powershell
.venv\Scripts\python tools\import_questions.py 题库.xlsx --project-id 1 --subject-id 6 --commit
```

## 数据备份与恢复

数据库、图片、原始导入文件和配置都位于 `data` 目录。手工备份前先关闭软件，再完整复制整个目录；恢复时关闭软件并用备份目录替换当前 `data`。

通用环境变量为 `STUDY_DATA_DIR`、`STUDY_DB_NAME`、`STUDY_SECRET` 和 `STUDY_MAX_UPLOAD_MB`。旧的 `H3CSE_DATA_DIR`、`H3CSE_SECRET` 仍兼容，现有数据库文件名不会被强制更改。

## 公开仓库与本地题库

`data`、`题库` 和 `backups` 均被 Git 忽略，真实题目、学习记录和导入原件可以继续保留在本机。公开仓库不包含数据库或示例题；全新克隆首次启动时会自动创建零题目的数据库和默认知识结构。

提交或推送前运行：

```powershell
check-public-repo.bat --no-pause
```

检查器只读取 Git 索引与完整历史，不遍历或修改本机 `data`。GitHub Actions 会对每次推送和 Pull Request 执行相同检查。**不要运行 `git clean -fdx`**，因为 `-x` 会删除包括本地题库在内的忽略文件。

首次公开发布、仓库外备份和干净克隆验收步骤见 [公开仓库发布检查清单](docs/public-release-checklist.md)。

## 开发与测试

```powershell
.venv\Scripts\python -m pip install -r requirements.txt
.venv\Scripts\python -m unittest discover -s tests -v
```

学习统计和准备度只用于个人决策，不代表任何机构的官方考试标准。
