# 我爱学习 v2.0

“我爱学习”是一款本地优先的多项目题库、练习、复习与进度管理工具。本次重建 `v2.0`，集中完成全局界面升级，并保留现有题库、学习记录和导入导出格式兼容。

## 本次更新

- 建立蓝靛色语义设计系统，统一浅色、深色和跟随系统主题下的排版、间距、控件与状态反馈。
- 桌面侧栏按“学习、内容、系统”分组；手机端固定“今日、练习、复习、进度、更多”五个入口。
- 首页改为行动导向，优先展示继续会话、到期复习、今日计划或自由练习，减少重复统计和装饰性信息。
- 练习、复习与模拟考试采用更专注的答题布局，暂停、恢复、终止、计时和保存状态更容易识别。
- 题库与任务列表继续保持高信息密度，同时改善筛选、分页、批量操作和手机端结构化列表。
- 知识树统一为“科目 → 章节 → 知识点”三级嵌套卡片，通过缩进、引导线、图标和一致操作区明确层级。
- 项目管理、数据管理、设置以及导入与导出页面统一表单和危险操作层级。
- 保持 Windows Portable 的 GUI / Web 双模式和 Linux Docker Compose 部署兼容。

## 安装与升级

1. 下载 `I-Love-Learning-Portable.zip` 并完整解压。
2. 双击 `I-Love-Learning.exe`。
3. 首次使用选择“软件内打开（GUI）”或“浏览器打开（Web）”。
4. 升级已有版本时，先停止旧程序，并保留原完整数据目录及 `.portable-launcher.json`。

软件没有账号与公网访问保护，请只在自己的电脑或可信局域网中使用，不要配置公网端口映射。GUI 模式需要 Microsoft Edge WebView2 Runtime。

## 数据兼容

- 不内置任何真实题库、学习记录或导入原件。
- 现有数据库、题目图片、答题记录和项目配置保持兼容。
- 分享题库请使用软件内“导入与导出”的 ZIP 分享包，不要复制运行中的 SQLite 数据库。
