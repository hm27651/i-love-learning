# 全局 UI 重构

## Goal

将“我爱学习”重构为沉静专注、扁平编辑式的个人学习工具。在不改变业务功能、URL、数据库或学习数据的前提下，降低视觉噪声、强化信息层级，并同时改善桌面、390px 手机、浅色与深色体验。

## Background

- 当前界面有大量同权卡片、边框、阴影、颜色和眉题，首页与手机端尤其密集。
- 应用使用 Flask、Jinja、原生 CSS/JavaScript，并由 Windows Portable GUI/Web 与 Linux Docker 共用页面。
- 已确认保留桌面左侧栏、手机底部导航、内置 SVG、主题切换和现有模块开关。

## Requirements

- 建立蓝靛品牌色、冷中性色和完整语义状态色的浅/深色设计 Token。
- 使用系统原生中文字体栈、12/14/16/20/28/36px 字级、等宽数字和 4/8px 间距体系，不引入外部字体或前端依赖。
- 以排版、网格、留白和分隔线建立层级，减少普通内容的卡片、边框与阴影；每个视区最多一个主要操作。
- 桌面导航按“学习、内容、系统”分组；手机固定“今日、练习、复习、进度、更多”，且尊重项目模块开关。
- 首页首先显示下一项学习行动，优先级为活动会话、到期复习、今日计划、自由练习；趋势、计划和准备度改为可跳转摘要。
- 练习、复习和模拟使用专注答题布局，保留现有暂停、继续、终止、计时、自动保存和判分规则。
- 桌面题库、审核和任务列表保持紧凑表格；手机改为结构化记录列表，不依赖横向滚动。
- 知识树、导入导出、项目、数据和设置页面使用统一页面头、工具栏、分组表单、状态反馈和危险操作样式。
- 仅保留能指导学习决策的趋势与弱项图表，其他统计改为数字摘要。
- 交互动效限制在 150–200ms，并支持 `prefers-reduced-motion`。
- 正文和控件达到 WCAG AA；焦点可见，状态不只依赖颜色，触控目标至少 44px。

## Acceptance Criteria

- [ ] 1440px 桌面和 390px 手机均能完成所有主要学习与管理流程，主要操作无横向滚动。
- [ ] 浅色、深色、跟随系统和减少动态效果模式显示一致且可读。
- [ ] 首页首屏可以明确识别唯一的下一项学习行动，次级统计不再争夺视觉焦点。
- [ ] 桌面侧栏分组、收起状态、当前项与模块显隐正确；手机“更多”能访问所有其余启用模块。
- [ ] 练习、复习、模拟的提交、解析、自评、暂停、继续、终止、计时与保存行为无回归。
- [ ] 题库筛选、批量审核、分页、知识结构管理、导入导出和危险操作均保持可用。
- [ ] 组件使用语义 Token，不再依赖大量页面级硬编码颜色和追加式 `!important` 覆盖。
- [ ] 键盘可完成导航、表单、答题和弹窗操作，焦点顺序与视觉顺序一致。
- [ ] 完整自动测试、Python 语法检查、公开仓库检查和 Portable 烟测通过。

## Out of Scope

- 不修改数据库、业务接口、URL、导入导出格式或学习记录。
- 不增加 Node、前端框架、外部 CDN、在线字体、自由拖拽首页或用户自定义组件。
- 不重新设计业务规则，也不修改 Windows Portable 与 Linux Docker 的部署边界。
